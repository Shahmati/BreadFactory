# BreadFactory
ИС "Хлебобулочный завод"
Предназначена для автоматизации процесса управления пользователями.
Реализована авторизация, проверка с помощью интерактивной капчи. Для пользователей с ролью "Администратор" реализована возможность добавлять, редактировать, удалять, разблокировать пользователей.
Для запуска приложения необходимо запустить BreadFactory.exe
